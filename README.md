# Benzinverbrauch – Minimalistische Verbrauchs- und Kostenberechnung

Eine kleine, schnelle und komplett clientseitige Web‑App zur Berechnung von
Benzinverbrauch, Kosten pro Kilometer und weiteren Fahrdaten.  
Die App läuft vollständig im Browser, speichert Werte lokal und benötigt
keine Installation.

👉 **Sofort ausprobieren über GitHub Pages:**  
https://lastcaballero.github.io/Benzinverbrauch/

---

## 🚀 Features

- **Verbrauchsberechnung**  
  Berechnet automatisch Verbrauch pro 100 km, Kosten pro km und Gesamtkosten.

- **Lokale Speicherung**  
  Alle Eingaben werden im Browser gespeichert (LocalStorage).

- **Modulare Struktur**  
  Klare Trennung von UI, Logik, Berechnungen und Storage.

- **Sofort nutzbar**  
  Einfach `index.html` öffnen – keine Abhängigkeiten, kein Build‑Prozess.

- **Responsives Layout**  
  Dank sauberer CSS‑Struktur (base, components, layout).
